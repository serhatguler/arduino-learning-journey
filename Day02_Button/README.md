**# Day 02 - Button Input**



**## Objective**

Learn digital input handling using a push button with Arduino.



**## What I did**

- Connected a push button to Arduino
- Read digital input using digitalRead()
- Used internal pull-up resistor
- Controlled LED using button input



**## Key Concepts**

- digitalRead()
- INPUT_PULLUP
- Button debouncing (basic)

## Notes
This project focuses on reliable digital input handling, which is critical in embedded systems.
